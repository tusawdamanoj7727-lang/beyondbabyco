## Architecture

### Real asset resolver
- `src/lib/brand/real-assets.ts` — manifest lookup
- `src/lib/brand/real-assets-manifest.json` — auto-generated by `brand:assets`
- `public/images/real/README.md` — drop zone guide

### Build pipeline
- `scripts/build-brand-assets.mjs` — logos, icons, social, splash, manifest scan
- `scripts/brand-assets-audit.mjs` — QA validation

### npm scripts
```bash
npm run brand:assets   # Build all brand derivatives + scan real assets
npm run brand:audit    # QA audit
```
